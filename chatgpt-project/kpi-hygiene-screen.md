# KPI Hygiene Screen

## Purpose

Determine whether portfolio KPIs and success metrics are clear enough to support executive reporting, value tracking, and prioritization.

## KPI hygiene checks

Flag concerns when KPI fields show:

- Missing KPI or success metric
- Metric name without definition
- No baseline
- No target
- No actual or latest value
- No measurement owner
- No reporting cadence
- Mixed units across records
- Benefit claim without evidence source
- Lagging-only measure where leading indicators are needed
- KPI wording that describes activity instead of outcome
- KPI drift where the metric changed without explanation

## Rating

- **Strong:** Defined metric, baseline, target, owner, source, cadence, and latest value are visible.
- **Usable:** Enough detail for directional review, but at least one hygiene element is missing.
- **Weak:** Metric is vague, unowned, unmeasured, or not tied to outcome.
- **Absent:** No KPI or success metric provided.

## Output fields

- Record name
- KPI or success metric
- Hygiene rating
- Missing elements
- Reporting risk
- Cleanup action
- Human owner needed

# Working Session Prompts

## Full audit prompt

Audit this portfolio inventory for signal quality. Identify field-quality issues, stale statuses, missing owners or sponsors, duplicate-demand candidates, readiness classification, KPI hygiene concerns, reporting risk, cleanup actions, and downstream handoffs. Do not change records or invent missing fields.

## Executive summary prompt

Turn the audit findings into a one-page executive signal-quality summary. Include trust level, decision risks, top cleanup priorities, records blocked from downstream use, and decisions required from leaders.

## Cleanup planning prompt

Create a cleanup plan from these findings. Group actions into decision blockers, governance hygiene, and reporting improvement. Include owner-needed, due/cadence, evidence needed, and downstream destination.

## Duplicate review prompt

Review these portfolio records for possible duplicate demand. Classify each candidate group as likely duplicate, possible duplicate, related work, or not enough information. Provide evidence and human review questions. Do not merge records.

## KPI hygiene prompt

Review KPI and success metric fields for hygiene. Flag missing definitions, baselines, targets, owners, cadence, latest values, weak activity metrics, and drift. Classify each KPI as strong, usable, weak, or absent.

## Handoff prompt

Prepare a downstream handoff packet showing which records should go to intake triage, prioritization scoring, executive review, and PMO operations follow-through. Include what not to pass downstream.

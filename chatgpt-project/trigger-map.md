# Trigger Map

## Use this module when the user says

- Audit this portfolio spreadsheet.
- Clean up this project list.
- Tell me whether this portfolio report is trustworthy.
- Identify stale projects, missing owners, or duplicate demand.
- Check whether this Jira export is ready for portfolio review.
- Review this Smartsheet table before an executive meeting.
- Find status, ownership, readiness, and KPI issues.

## Run a full audit when inputs include

- Multiple initiatives, projects, epics, programs, or demand items
- Status fields
- Owners or sponsors
- Dates or milestones
- Readiness, phase, or lifecycle indicators
- KPI, benefit, measure, or success metric fields

## Run a limited audit when inputs lack

- Dates: skip stale-status conclusions and flag date gaps.
- Owners or sponsors: run ownership gap screen only.
- KPI fields: flag KPI absence but do not infer KPI quality.
- Descriptions: duplicate review will be weaker; state confidence limitations.

## Do not use this module for

- Official financial validation
- Legal, compliance, privacy, security, audit, accounting, or HR determinations
- Live-system updates
- Funding approval
- Prioritization scoring
- Project charter creation
- Business case creation
- Release approval
